# Real-Time-Sign-Language-Recognition
Real time Indian Sign Language recognition system using Neural Network

## Description
A real time Indian Sign Language recognition system 
that detects hand gestures and converts to text and speech.

## Tech Stack
- Python
- TensorFlow
- OpenCV
- pyttsx3

## Duration
January 2024 - April 2024

## Features
- Detects hand gestures in real time
- Converts gestures to text
- Converts text to speech

## How to Run
1. Clone the repository
2. Install required libraries
3. Run main.py

## Libraries Required
pip install tensorflow opencv-python pyttsx3
